#Barangay Transaction Management System

---
  ``` shell
  Instructions:

  1. Download the project repository
       • $ git clone https://github.com/ninoypaninoy/barangay-transaction/

  2. Go to the root directory of the repository
       • $ cd barangay-transaction

  3. Install missing gems
       • $ bundle install
       • $ rake db:migrate

   4. Run the server
       • $ rails s
       • Then open a browser and enter localhost:3000/sign_up
       
   5. Documentation
       • See the full documentation in /barangay-transaction/Documentation/BarangayTransactionManagementSystem.pdf
  ```
